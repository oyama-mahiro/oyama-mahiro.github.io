4、const指针

​ const修饰的变量为只读属性，不能直接修改该变量的值

​ const指针

​ (1)const 数据类型 *指针变量名;

​ 数据类型 const *指针变量名;

​ const修饰是指针指向的内容，不能修改指针指向的数据，但是可以修改指针变量中的地址值

​

​ (2)数据类型 * const 指针变量名；

​ const修饰的指针变量p，不能修改指针变量的值，但是可以修改指针指向内容的值

（3）const 数据类型 * const 指针变量名；

​ const同时修饰指针变量和指针指向内容，两者都不能修改
